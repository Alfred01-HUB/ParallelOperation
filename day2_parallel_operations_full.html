Resources
MDNReference
Promise.allSettled() — MDN
Returns a result object per promise ({status, value/reason}) rather than rejecting. The safe alternative to Promise.all for partial failures.
developer.mozilla.org/.../Promise/allSettled
MDNReference
Promise.race() — MDN
Settles with whichever promise settles first — useful for timeouts. Pair with the Promise.any page for a complete picture.
developer.mozilla.org/.../Promise/race
